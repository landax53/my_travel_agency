Qui sommes-nous\u00a0?<\/h2>\n\n\n\n

  • Nous stockons vos donn\u00e9es de mani\u00e8re s\u00e9curis\u00e9e ;<\/li>

  • Nous vous laissons mettre \u00e0 jour, supprimer et consulter vos donn\u00e9es ;<\/li>

    1. Qui est responsable de vos donn\u00e9es personnelles ?<\/h3>\n\n\n\n

    My Travel Agency est responsable du traitement de vos donn\u00e9es \u00e0 caract\u00e8re personnel.     2. Quelles donn\u00e9es collectons-nous ?<\/h3>\n\n\n\n

    Les donn\u00e9es \u00e0 caract\u00e8re personnel sont des donn\u00e9es permettant de vous identifier soit directement (nom, pr\u00e9nom\u2026), soit indirectement (num\u00e9ro de t\u00e9l\u00e9phone, num\u00e9ro de dossier\u2026).<\/p>\n\n\n\n

    • donn\u00e9es commerciales (historique d\u2019achats\u2026)<\/li>
    • profil commercial (produits pr\u00e9f\u00e9r\u00e9s, modes de communication pr\u00e9f\u00e9r\u00e9s\u2026)<\/li>
